聊天系统 API 接口文档

注意：所有接口都需要JWT认证，任何已登录的用户都可以访问聊天功能。 请先登录获取JWT token，然后在请求头中包含：Authorization: Bearer <your_token>

🔐 认证

POST /api/token/
获取JWT token
POST /api/token/refresh/
刷新JWT token

🏠 房间管理

GET /api/chat/rooms/
获取房间列表
POST /api/chat/rooms/
创建新房间
GET /api/chat/rooms/{id}/
获取房间详情
POST /api/chat/rooms/{room_id}/join/
加入房间
POST /api/chat/rooms/{room_id}/leave/
离开房间
GET /api/chat/rooms/{room_id}/members/
获取房间成员

💬 消息管理

GET /api/chat/rooms/{room_id}/messages/
获取房间消息（支持分页）
POST /api/chat/rooms/{room_id}/messages/
发送消息
POST /api/chat/messages/{message_id}/read/
标记消息已读
DELETE /api/chat/messages/{message_id}/delete/
删除消息（软删除）

🔔 通知管理

GET /api/chat/notifications/
获取通知列表
POST /api/chat/notifications/{notification_id}/read/
标记通知已读

🔌 WebSocket 连接

WS ws://localhost:8000/ws/chat/{room_id}/
WebSocket实时通信连接

📝 请求示例

创建房间

POST /api/chat/rooms/
Content-Type: application/json
Authorization: Bearer <your_jwt_token>

{
    "name": "新房间",
    "description": "房间描述",
    "room_type": "public",
    "max_members": 50
}

发送消息

POST /api/chat/rooms/1/messages/
Content-Type: application/json
Authorization: Bearer <your_jwt_token>

{
    "content": "大家好！",
    "message_type": "text"
}

WebSocket消息格式

// 发送消息
{
    "type": "chat_message",
    "content": "消息内容",
    "message_type": "text"
}

// 接收消息
{
    "type": "chat_message",
    "message": {
        "id": 1,
        "sender": {
            "id": 1,
            "username": "developer"
        },
        "content": "消息内容",
        "created_at": "2024-01-01T12:00:00Z"
    }
}

  1. 使用developer账户登录获取JWT token
  2. 使用token访问API接口
  3. 创建房间并加入
  4. 发送消息并建立WebSocket连接
  5. 处理实时消息和通知
