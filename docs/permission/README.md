# 权限管理文档

本文件夹包含权限管理相关的所有文档。

## 文档列表

- `PERMISSION_GUIDE.md` - 权限管理使用指南
- `PERMISSION_SUMMARY.md` - 权限管理功能总结

## 功能说明

权限管理系统提供以下功能：
- 用户权限管理
- 角色权限分配
- 菜单权限控制
- 权限日志记录

## 使用指南

1. **基础使用**：查看 `PERMISSION_GUIDE.md`
2. **功能总结**：查看 `PERMISSION_SUMMARY.md`

- 权限数据存储在 `permission_app` 应用中
- 支持细粒度权限控制
- 提供权限验证工具函数
- 集成用户认证系统
